# 📚 Library Management System

## 📌 Project Description
This is a Library Management System built using Spring Boot.
It provides REST APIs to manage books in a library.

## 🚀 Features
- Add new book
- View all books
- View book by ID
- Update book details
- Delete book

## 🛠 Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven

## 📂 Project Structure
- controller → Handles REST API requests
- service → Business logic layer
- repository → Database interaction
- entity → JPA entity classes

## ▶ How to Run
1. Clone the repository
2. Open in Eclipse / IntelliJ
3. Configure database in `application.properties`
4. Run as Spring Boot Application
